**GitHub Copilot code reviews are now coming to Azure Repos — and if your team is still on Azure DevOps, this is worth a few minutes of your time.**

Microsoft just announced a limited public preview of Copilot code reviews inside Azure Repos pull requests. Same engine that's been live on GitHub.com — now surfacing directly in your Azure DevOps PR workflow.

Here's the short version of what matters:

- **No new UI to learn** — request a Copilot review from the Reviewers section, just like you'd ping a colleague
- **Three-tier enablement** — org → repo → user, so you control exactly where it's active
- **Billing via AI credits** on your Azure subscription (1 credit = $0.01 USD), visible in Azure Cost Management
- **Best fit for code-centric repos** — Dataverse plugins (C#) and PCF controls (TypeScript) are the obvious first targets
- Canvas apps and Power Automate flows are **not in scope** yet

Preview limits are reasonable for most Power Platform ALM workflows (100 files per PR, 10 GB repo max, 5 concurrent reviews per org).
